# programacao-c

A C programming repository for learning and exercises.

## Setup

- **Language:** C (compiled with GCC 14)
- **Build System:** Make (`Makefile`)
- **Workflow:** `Start application` — runs `make run` (compiles and executes `main.c`)

## Project Structure

- `main.c` — Entry point / starter C program
- `Makefile` — Build rules: `make` compiles, `make run` compiles and runs, `make clean` removes binaries

## Running

The workflow automatically compiles and runs `main.c`. To add new C programs, either extend `main.c` or add new source files and update the `Makefile` accordingly.
